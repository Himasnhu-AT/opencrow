# Orders

This document explains how to view and manage your orders in the ShopDemo application.

## Viewing Orders

The main page of the application displays a list of your recent orders. You can see the status, total, and items for each order.

## Viewing Order Details

To see more details about a specific order, you can ask the AI assistant:

> "Show me the details for order ord-001"

The assistant will then display the full details of the order.

## Cancelling an Order

You can cancel an order using the AI assistant. For example:

> "Cancel my order ord-001"

The assistant will then confirm that the order has been cancelled.
