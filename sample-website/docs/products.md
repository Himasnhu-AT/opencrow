# Products

This document explains how to interact with products in the ShopDemo application.

## Viewing Products

The main page of the application displays a grid of featured products. You can scroll through this grid to see the available products.

## Searching for Products

You can use the AI assistant to search for products. For example, you can say:

> "Find wireless headphones"

The assistant will then display a list of products that match your search query.
